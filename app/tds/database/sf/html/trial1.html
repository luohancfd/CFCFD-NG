Background

This is a simple Browser created to query a PostgreSQL Database and display the query results of the shock metadata. I'm trying my best to improve the user interface of this browser, so please wait patiently, it is under construction at the moment.

Instructions

In order to do a search, you are required to key in a query in the box provided and press submit to get the results. EASY!

Keywords : SELECT, FROM and WHERE
Table Name : shot_descriptions (*all must in LOWERCASE)
Search using: shot_number (*all must in LOWERCASE)
Example of a query: "SELECT * FROM shot_descriptions WHERE shot_number=7000"
Note: * stands for all available information

More details...

You can either search for all items in the specific file or do a search for selected items given in the table below:

facility_name shot_number basic_file_name date_string project
blame reservoir nozzle shocktube notes
diaphragm 1 diaphragm 2 diaphragm 3 driver 0 driver 1
driver 2 condition acceltube dumptank jackoff

Range for shot_number: anything greater than 3720! (minimum)
Yup! You're right! There is limited shock tunnel files available for searching, so just stick on to the range given above and most likely you will be able to get the results you requested for. GOOD LUCK!
The maximum range for searching is not given simply because the experiment is an on going stuff!
