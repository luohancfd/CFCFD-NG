INSTRUCTIONS


In order to do a search, you are required to key in a SQL query in the textbox provided. Remember to press the SUBMIT QUERY button to get the results and press the CLEAR button to clear the textbox. EASY! And if you require raw data to be included into your results, just simply put a tick in the box provided at the end of this page. Done!

An SQL statement should looks something like that: "SELECT ______ FROM table name WHERE ______"

The following will show you how to form a proper SQL statement:

Keywords : SELECT, FROM and WHERE
Table Name : shot_descriptions (all must in LOWERCASE and remember included a "s")
Search using : e.g. shot_number (all must in LOWERCASE)
When combined : "SELECT * FROM shot_descriptions WHERE shot_number=8000"

Note: * stands for all available information

DIFFERENT QUERY TYPES


In fact, SQL syntax is kinda of clear, simple and reasonably easy to understand if you have some relevant SQL knowledge. But it can be fairly nasty and a nightmare if you do not understand them at all. Hence, please use "Quick Search" if you are new to SQL.

If you are: 1)One that had learned SQL before but long forgotten. 2) One that has some knowledge of SQL but not very sure about the types of query you can make to perform a search this site. You are at the RIGHT section! This section will show you some useful queries that you can use for certain situations.

Qn 1 : If I only need project, blame, date and shot number for shot number = 7966, how should I type?
Ans : SELECT project, blame, date_string, shot_number FROM shot_descriptions WHERE shot_number=7966
Qn 2 : If I need to look for the experiments that I done before such as project and shot number, how should I type?
Ans : SELECT project, shot_number FROM shot_descriptions WHERE blame ~ 'your initial'
Qn 3 : If I need find everything related to scramjet project, how should I type?
Ans : SELECT * FROM shot_descriptions WHERE project ~ 'scramjet'
Qn 4 : If I need to look for everything between shot number 8000 to 8100, how should I type?
Ans : SELECT * FROM shot_descriptions WHERE (shot_number > 8000 and shot_number < 8100)

Note: Please note that the searching time is very much dependent on the number of stuffs that you search for. So please wait patiently if it takes quite a while.
