Welcome to the Centre for Hypersonics' Experimental Facilities Data Manager!

This is essentially a site developed for both searching run description text files obtained from T4 facility and for uploading run description and flow image data from all experimental facilities. This database is essentially a mini library for T4 run description text files. These files contains the experimental conditions of each shot. This database is especially useful if you need to retrieve all the run description data from all the shots you have done. It is able to call out an individual record of your request at a time as well. A search can be made by first selecting a search type that suit your purpose from the MENU above (green square buttons). It consists of two different types of searching method. Please read the following two sections for greater details on each searching capability. Additionally, you can record and save the experiment details of both run description and flow image data using one of the upload page from the MENU above (yellow square buttons).

Note: If you require graphical plots for your numerical raw data, please use the Td Browser instead.

QUICK SEARCH PAGE


This is the Quick Search site, where you can perform a quick search (obviously) by selecting a field, such as project or shot number and type in relevant search words. Besides, you can also choose the number of information you want to show for a particular search. For more details about performing a search, please consult the Instruction Menu located in the same page (Quick Search Site). For other search options, please select from the search menu bar (the ones with green buttons) above.


MANUAL SEARCH PAGE


This is the Manual Search site, where you have to key in a SQL statement in order to perform a search. "Manual Search" is not highly recommended especially for people without technical grounding. However, it is provided for users who know SQL well enough and need an alternative searching method. For detail information regarding how to perform a search and different query types, please refer to the Instruction Menu or click here for more help. For other search options, please select from the search menu bar (the ones with green buttons) above.

Notes: Quick Search is highly recommended!

 

RUN DESCRIPTION DATA UPLOAD PAGE


This is a form designed specifically for run description text. You are required to fill in the experimental details of your shot into the respective boxes provided and a short descriptions on the experiment. For flow image data upload form, please select from the search menu bar (the ones with yellow buttons) above.
 

FLOW IMAGE DATA UPLOAD PAGE


This is a form designed specifically for flow image data. You are required to fill in the experimental details of your optical diagnostics into the respective boxes provided and a short descriptions on the experiment. For run description data upload form, please select from the search menu bar (the ones with yellow buttons) above.
