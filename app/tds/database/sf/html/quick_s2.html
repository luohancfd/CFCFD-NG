QUICK SEARCH PAGE


This is the Quick Search site, where you can perform a quick search (obviously) by selecting a field, such as project or shot number and type in relevant search words. Besides, you can also choose the number of information you want to show for a particular search. For more details about performing a search, please consult the Instruction Menu below. For other search options, please select from the search menu bar (the ones with green buttons) above.

You can click "Home" at the top right hand corner to return to the homepage. And you can easily return back to this page by clicking "Back" provided in the Results Page.

INSTRUCTIONS


To perform a search, first you have to select the information you require by clicking the checkbox provided. Next, select your desired searching field, such as project, blame or shot number. Followed by, typing in a relevant search word, for example 7996 for shot number. Lastly, press the SEARCH button to begin your results. SIMPLE! And of course, to clear the textbox, simply press the CLEAR button. Additionally, if you require raw data to be included into your results, just simply put a tick in the box provided. Besides, you can choose the 2 different types of display format for your results by checking the box provided as well. Done!
 

TWO TYPES OF DISPLAY FORMAT

Type 1 : Table format where columns=fields and rows=records. This is especially useful if your results is NOT too huge.
For an example : If you typed in project, blame and shot_number, it will show:
project blame shot_number
scramjet SF/PJ 1234

Type 2 : Displays each result individually. This is especially useful if your results is real HUGE.
For an example : If you typed in *, which indicates show all fields, it will show:
Field Names Values
project scramjet
blame SF/PJ
shot_number 1234
... ...
