QUICK SEARCH SITE


This is the Quick Search site, where you can perform a quick search (obviously) by selecting a field, such as project or shot number and type in relevant search words. Besides, you can also choose the number of information you want to show for a particular search. For more details about performing a search, please consult the Instruction Menu below. For other search options, please select from the search menu bar (the ones with green buttons) above.

You can click "Home" at the top right hand corner to return to the homepage. And you can easily return back to this page by clicking "Back" provided in the Results Page.

INSTRUCTIONS


To perform a search, first you have to select the information you require by clicking the checkbox provided below. Next, select your desired searching field, such as project, blame or shot number. Followed by, typing in a relevant search word, for example 7996 for shot number. Lastly, press the SEARCH button to begin your results. SIMPLE! And of course, to clear the textbox, simply press the CLEAR button. Also, if you require raw data to be included into your results, just simply put a tick in the box provided at the end of this page. Done!

EXAMPLES OF SEARCH


If you are still uncertain about how can you perform a search at this page, don't worry! The following examples are provided to guide you through the initial stage. After this short tutorial, you will find searching so easy that you will start to laugh!

Qn 1 : If I only need project, blame, date and shot number for shot number = 7966, how should I type?
Ans : SELECT project, blame, date_string, shot_number FROM shot_descriptions WHERE shot_number=7966
Qn 2 : If I need to look for the experiments that I done before such as project and shot number, how should I type?
Ans : SELECT project, shot_number FROM shot_descriptions WHERE blame ~ 'your initial'
Qn 3 : If I need find everything related to scramjet project, how should I type?
Ans : SELECT * FROM shot_descriptions WHERE project ~ 'scramjet'
Qn 4 : If I need to look for everything between shot number 8000 to 8100, how should I type?
Ans : SELECT * FROM shot_descriptions WHERE (shot_number > 8000 and shot_number < 8100)

Note: Please note that the searching time is very much dependent on the number of stuffs that you search for. So please wait patiently if it takes quite a while.
