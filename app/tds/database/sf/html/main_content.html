Background


This is a simple Browser created to query a PostgreSQL Database and display the query results of the shock metadata. I'm trying my best to improve the user interface of this browser, so please wait patiently, it is under construction at the moment.
 

Instructions


In order to do a search, you are required to key in a query in the box provided and press submit to get the results. EASY!

:e.g. shot_number (all must in LOWERCASE)
:"SELECT * FROM shot_descriptions WHERE shot_number=7000"

facility_name shot_number basic_file_name date_string project

Note: All the above are case sensitive, make sure you check!

Range for shot_number: anything greater than 3720! (minimum)
